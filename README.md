# スクロールに応じてグラデーションが切り替わるアニメーション
## Description
Appleの新Macbook Proのページにあるスクロールに応じて文字のグラデーションが変わるアニメーションを実装した練習。
## How to view
1. npm i
2. gulp（後々修正予定）
3. index.html, global.scss, common.tsを再度保存
